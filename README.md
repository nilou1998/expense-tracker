# Expense Tracker App

A simple expense tracker built with React, TypeScript, Tailwind, and Vite.

## Features (current stage)

- Static UI layout
- Transaction list with mock data
- Basic state management
- Type-safe components
- Responsive UI structure

## Tech Stack

- React
- TypeScript
- Tailwind CSS
- Vite

## Current Status

This project is in early development stage.  
Next steps include adding transaction form, state updates, and persistence.
